# Quizzy

A MCQ Quiz Platform


## Installation

```bash
pip install -r requirements.txt
```

## Usage

Open app.py and setup database 
```python
host="hostname",
user="username",
password="password",
database="db_name"
```

Run the program using 
```bash
python app.py or flask run
```
